# DtMoney

Is application to manager off deposits and withdraws finance.

### Technologies used

> Typescript  
> ReactJs  
> Styled-components  
> MirageJs

## To run project

In the project directory, you can run:

`yarn start` to run in development mode.

`yarn build` to create the build for production project.
